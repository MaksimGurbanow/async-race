# Async-race

### Description:
This project was inspired to enhance my skills in asyncronous programming. Here I implemented the game offering garage page with cars. The cars are taken from the server, which also need to run locally on your computer if you want to test this application. Here is the link to server [repo](https://github.com/mikhama/async-race-api). The project was inspired by rolling Scopes School.
